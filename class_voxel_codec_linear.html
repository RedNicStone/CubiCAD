this class represents a linear voxel codec, stored like this: n 1 2 3 data [ <attribute0>, <attribute0>, <attribute0>, <attribute1>, <attribute1>, <attribute1>, ..., <attributeN>, <attributeN>, <attributeN> ]

this implementation is probably slower than "VoxelCodecShuffle", but the attributes are kept separate which makes them easy to stream
